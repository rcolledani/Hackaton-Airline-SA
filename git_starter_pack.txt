Se déplacer vers le folder de votre choix :
        - cd .../Hackaton
On initialise l'environnement git sur le folder où on est actuellement
	-git init
On défini la "remote" connexion vers le repository. Le nom de la connexion est "origin"
	-git remote add origin https://github.com/youssefOud/Hackaton-Airline-SA.git
On pull (importe) la branche "main" du repository avec la connexion "remote" qu'on a défini dans la commande précédente
	-git pull origin main

Pour des raisons organisationnelles on va pas travailler sur la branche main directement mais sur des branches à part en fonction
de ce qu'on veut faire.
Pour créer une nouvelle branche :
	-git checkout -b nameOfNewBranch
Pour switcher de branche: git checkout main

On modifie le contenu de la branche ou on y rajoute de nouveaux fichiers etc..
On sélectionne les trucs qu'on veut ajouter (ici le "." veut dire qu'on rajoute tout)
	-git add .
On commit les changements, en nommant le commit de manière explicite
	-git commit -m "feature/FeatSentiment analysis model v2.1"
On push dans la branche de notre choix les modifications (de préférence, ne pas push sur la 'branch' main)
	-git push -u origin nameOfNewBranch

On peut aussi importer la dernière version du code depuis git (ici on importe depuis la branch 'main')
	-git pull origin main

Pour voir les 'branch' dispos en local :
	-git branch
Pour voir les 'branch' dispo en remote:
	-git branch -r
